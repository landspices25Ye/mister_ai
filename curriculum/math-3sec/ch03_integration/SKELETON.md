# هيكل الفصل الثالث: التكامل غير المحدود
> **الحالة:** هيكل تفصيلي
> **الزمن الإجمالي:** 3-4 حصص

---

## الدرس 3.1: مفهوم التكامل

**الأسئلة التأسيسية:**
- ما العملية المعاكسة للاشتقاق؟ → **التكامل**
- إذا كانت F'(x) = f(x)، نقول F هي **مقابل (antiderivative)** لـ f
- التكامل يعطي "عائلة" من الدوال (لأنه نضيف ثابت C)

**التمثيل:**
∫ f(x) dx = F(x) + C

حيث C = ثابت التكامل (يمثل الإزاحة الرأسية)

---

## الدرس 3.2: قواعد التكامل الأساسية

**الجدول الذهبي:**

| f(x) | ∫ f(x) dx |
|---|---|
| 0 | C |
| 1 | x + C |
| x^n | x^(n+1)/(n+1) + C (n ≠ -1) |
| 1/x | ln\|x\| + C |
| e^x | e^x + C |
| sin x | -cos x + C |
| cos x | sin x + C |
| sec²x | tan x + C |
| csc²x | -cot x + C |
| sec x tan x | sec x + C |
| csc x cot x | -csc x + C |
| a^x | a^x / ln a + C |

**قواعد الدمج:**
- ∫ [f(x) + g(x)] dx = ∫ f(x) dx + ∫ g(x) dx
- ∫ c·f(x) dx = c·∫ f(x) dx

**مثال:**
∫ (3x² + 2x - 1) dx = x³ + x² - x + C

---

## الدرس 3.3: التكامل بالتعويض (Substitution)

**الفكرة:** قاعدة السلسلة بالعكس.

إذا كان عندنا ∫ f(g(x))·g'(x) dx، نضع u = g(x)، فيصبح ∫ f(u) du

**الخطوات:**
1. اختر u (عادةً "الداخل")
2. احسب du = (du/dx) dx
3. عوّض في التكامل
4. ادمج بالنسبة لـ u
5. عوّض u بدلالة x
6. لا تنسَ +C

**مثال:**
∫ 2x·(x²+1)³ dx
- u = x²+1، du = 2x dx
- = ∫ u³ du = u⁴/4 + C = (x²+1)⁴/4 + C

**مثال آخر:**
∫ sin(x)·cos(x) dx
- u = sin x، du = cos x dx
- = ∫ u du = u²/2 + C = sin²(x)/2 + C

(أو بطريقة بديلة: = -cos²(x)/2 + C — متكافئتان!)
