## 📐 القواعد الثلاث الأساسية

### 🔹 القاعدة 1: قاعدة المجموع والفرق

$$[f(x) \pm g(x)]' = f'(x) \pm g'(x)$$

**اشتقاق كل حد على حدة، مع الإبقاء على الإشارات.**

**مثال:**
- `f(x) = sin(x) + x²` → `f'(x) = cos(x) + 2x`
- `g(x) = e^x - 5x³` → `g'(x) = e^x - 15x²`

> 💡 لا توجد "قاعدة الجداء البسيطة" بحد ذاتها (نحتاج قاعدة السلسلة لذلك)، لكن `c·f(x)` هي حالة خاصة منها.

---

### 🔹 القاعدة 2: قاعدة الجداء (Product Rule)

$$[f(x) \cdot g(x)]' = f'(x) \cdot g(x) + f(x) \cdot g'(x)$$

**"اشتقاق الأول × الثاني + الأول × اشتقاق الثاني"**

> 💡 **طريقة الحفظ:** "دُق الأول × الثاني + الأول × دُق الثاني" (دق = اشتقاق)

**مثال بسيط:** `f(x) = x² · (3x + 1)`
- `f'(x) = (x²)'·(3x+1) + x²·(3x+1)'`
- `= 2x·(3x+1) + x²·3`
- `= 6x² + 2x + 3x²`
- `= 9x² + 2x`

---

### 🔹 القاعدة 3: قاعدة القسمة (Quotient Rule)

$$\left[\frac{f(x)}{g(x)}\right]' = \frac{f'(x) \cdot g(x) - f(x) \cdot g'(x)}{[g(x)]^2}$$

**"اشتقاق البسط × المقام (بدون اشتقاق) − البسط × اشتقاق المقام، الكل على المقام تربيع"**

> 💡 **طريقة الحفظ:** "دُق ب · م − ب · دُق م، على م²"
> أو: "مشتقة البسط في المقام، ناقص البسط في مشتقة المقام، الكل على المقام تربيع"

**⚠️ انتبه للإشارة:** البسط في البسط فيه **ناقص** (ليس زائد)!

**مثال:** `f(x) = (x² + 1) / (x - 2)`
- البسط: `u = x² + 1` → `u' = 2x`
- المقام: `v = x - 2` → `v' = 1`
- نطبّق:
```
f'(x) = [(2x)·(x-2) - (x²+1)·1] / (x-2)²
      = [2x² - 4x - x² - 1] / (x-2)²
      = [x² - 4x - 1] / (x-2)²
```

## ✍️ أمثلة محلولة تفصيلية

### المثال 1: قاعدة الجداء مع دوال مثلثية

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = x² · sin(x)`

**الحل:**
- `u = x²` → `u' = 2x`
- `v = sin(x)` → `v' = cos(x)`
- نطبّق قاعدة الجداء:
```
f'(x) = (2x)·sin(x) + x²·cos(x)
f'(x) = 2x·sin(x) + x²·cos(x)
```

---

### المثال 2: قاعدة الجداء مع دالة أسية

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = e^x · (x³ + 1)`

**الحل:**
- `u = e^x` → `u' = e^x`
- `v = x³ + 1` → `v' = 3x²`
- نطبّق:
```
f'(x) = e^x·(x³+1) + e^x·3x²
f'(x) = e^x·[x³ + 1 + 3x²]
f'(x) = e^x·[x³ + 3x² + 1]
```

> 💡 **ملاحظة ذكية:** عند اشتقاق `e^x` تبقى كما هي — هذه ميزة فريدة للدالة الأسية الطبيعية.

---

### المثال 3: قاعدة القسمة (تطبيق شامل)

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = (2x + 3) / (x² - 1)`

**الحل:**
- `u = 2x + 3` → `u' = 2`
- `v = x² - 1` → `v' = 2x`
- نطبّق:
```
f'(x) = [2·(x²-1) - (2x+3)·(2x)] / (x²-1)²
      = [2x² - 2 - (4x² + 6x)] / (x²-1)²
      = [2x² - 2 - 4x² - 6x] / (x²-1)²
      = [-2x² - 6x - 2] / (x²-1)²
      = -2(x² + 3x + 1) / (x²-1)²
```

---

### المثال 4: مزيج من القواعد

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = (x² + 1) · (3x - 2) / (x + 5)`

**الحل:**
- أولاً: نعامل `(3x - 2) / (x + 5)` كدالة واحدة `g(x)`، ونحتاج `g'(x)`:
  - بتطبيق قاعدة القسمة:
  ```
  g(x) = (3x-2)/(x+5)
  g'(x) = [3·(x+5) - (3x-2)·1] / (x+5)²
        = [3x + 15 - 3x + 2] / (x+5)²
        = 17 / (x+5)²
  ```
- الآن نطبّق قاعدة الجداء على `f = (x²+1) · g`:
```
f'(x) = (x²+1)' · g(x) + (x²+1) · g'(x)
      = 2x · (3x-2)/(x+5) + (x²+1) · 17/(x+5)²
      = 2x(3x-2)/(x+5) + 17(x²+1)/(x+5)²
```
- نوحد المقام:
```
f'(x) = [2x(3x-2)(x+5) + 17(x²+1)] / (x+5)²
```

## ⚠️ أخطاء شائعة

| ❌ الخطأ | ✅ الصواب | لماذا؟ |
|---|---|---|
| `(x²·x³)' = 2x·3x²` | نستخدم `6x⁴` كاختصار (مشتقتها) | في البسط، لدينا ضرب، لا تطبيق قاعدة الجداء لزوم لها |
| نسيان الإشارة السالبة في القسمة | `f'g - fg'` في البسط | قاعدة القسمة فيها ناقص وليس زائد |
| نسيان تربيع المقام | `(g)²` في المقام | خطأ شائع عند الطلاب |
| `[(2x+1)/(x+3)]' = 1/1 = 1` | حساب فعلي | اشتقاق خارج القسمة لا يساوي اشتقاق البسط / اشتقاق المقام |

> 💡 **نصيحة:** في الاختبار، عندما تنسى قاعدة، **اشتقاقها** كدالة واحدة بعد تبسيطها قد يكون أضمن.

## 📝 تدريبات

### 🟢 مستوى 1
1. جد `f'(x)`:
   - `f(x) = (x² + 3x) · 5`
   - `f(x) = (x + 2) · (x - 5)`
   - `f(x) = 6x³ - 4x² + 2x - 7`

2. بسّط ثم اشتق:
   - `f(x) = x³ + 2x² · x`
   - `f(x) = (x+1)(x+2) - x²`

### 🟡 مستوى 2 (قاعدة الجداء)
3. جد `f'(x)`:
   - `f(x) = x · cos(x)`
   - `f(x) = (x² + 1) · e^x`
   - `f(x) = (2x + 1) · (x³ - x)`

4. جد ميل المماس لمنحنى `y = x · sin(x)` عند `x = π/2`

### 🟡 مستوى 2 (قاعدة القسمة)
5. جد `f'(x)`:
   - `f(x) = x / (x + 1)`
   - `f(x) = (x² - 1) / (x + 3)`
   - `f(x) = 2 / (x² + 1)`

6. جد `f'(2)` إذا كانت `f(x) = (3x - 1) / (x + 4)`

### 🔴 مستوى 3 (تحدّي)
7. جد `f'(x)` إذا كانت `f(x) = (x · cos x) / (sin x + 1)`
8. اشتق: `f(x) = (x+1)² · (x-1)³` (لاحظ: هناك طريقة مختصرة!)

**1.**
- `f(x) = 5x² + 15x` → `f'(x) = 10x + 15`
- `f(x) = x² - 3x - 10` → `f'(x) = 2x - 3`
- `f'(x) = 18x² - 8x + 2`

**2.**
- `x³ + 2x³ = 3x³` → `9x²`
- `x² + 3x + 2 - x² = 3x + 2` → `3`

**3.**
- `f(x) = x cos x` → `f'(x) = cos(x) - x sin(x)`
- `f(x) = (x²+1) e^x` → `f'(x) = (x²+1)e^x + 2x·e^x = e^x(x² + 2x + 1) = e^x(x+1)²`
- `f(x) = (2x+1)(x³-x)` → `f'(x) = 2(x³-x) + (2x+1)(3x²-1)`
  - `= 2x³ - 2x + 6x³ + 2x² - 3x² - x`
  - `= 2x³ + 2x³ - x² - 2x - x`
  - `= ...` → راجع الحل خطوة بخطوة

**4.** `y = x sin x` → `y' = sin x + x cos x`
   - عند `x = π/2`: `y' = 1 + (π/2)·0 = 1`
   - **ميل المماس = 1**

**5.**
- `f(x) = x/(x+1)`:
  - `u = x`, `u' = 1`; `v = x+1`, `v' = 1`
  - `f'(x) = [1·(x+1) - x·1] / (x+1)² = 1/(x+1)²`
- `f(x) = (x²-1)/(x+3)`:
  - `f'(x) = [2x·(x+3) - (x²-1)·1] / (x+3)²`
  - `= [2x² + 6x - x² + 1] / (x+3)²`
  - `= (x² + 6x + 1) / (x+3)²`
- `f(x) = 2/(x²+1)`:
  - `f'(x) = [0·(x²+1) - 2·2x] / (x²+1)² = -4x/(x²+1)²`

**6.** `f(x) = (3x-1)/(x+4)`:
   - `f'(x) = [3·(x+4) - (3x-1)·1] / (x+4)²`
   - `= [3x+12-3x+1] / (x+4)²`
   - `= 13 / (x+4)²`
   - عند `x=2`: `f'(2) = 13 / 36`

**7.** (تحدّي) استخدم الجداء والقسمة معاً:
- `f = (x cos x) / (sin x + 1)`
- `u = x cos x`, `u' = cos x - x sin x`
- `v = sin x + 1`, `v' = cos x`
- `f' = [u'v - uv'] / v²`
- `f' = [(cos x - x sin x)(sin x + 1) - x cos x · cos x] / (sin x + 1)²`

**8.** (الطريقة المختصرة):
- `f = (x+1)² · (x-1)³` (لكن لم نأخذ قاعدة السلسلة بعد!)
- الآن: `f = [(x+1)·(x-1)]² · (x-1) = (x²-1)² · (x-1)`
- نستخدم الجداء:
  - `u = (x²-1)²`, `v = x-1` — لكن اشتقاق `(x²-1)²` يحتاج سلسلة...
  - هذا تقدمة للدرس 1.4. **ارجع له بعد درس قاعدة السلسلة!**
</details>

## 🧪 اختبار قصير

**س1:** جد `f'(x)` إذا كانت `f(x) = (3x² + 1) · (x - 4)`

**س2:** جد `f'(x)` إذا كانت `f(x) = (x + 1) / (2x - 3)`

**س3:** بسّط ثم اشتق: `f(x) = (x+2)² - x·(x+4)`

**س4 (تطبيقي):** دالة إيراد شركة هي `R(x) = x · (50 - 0.5x)` حيث x عدد الوحدات.
- جد `R'(x)`
- ما الإيراد الحدي عند إنتاج 30 وحدة؟

<details>
<summary>إجابات</summary>

- **س1:** `u = 3x²+1`, `u' = 6x`; `v = x-4`, `v' = 1`
  - `f' = 6x(x-4) + (3x²+1) = 6x² - 24x + 3x² + 1 = 9x² - 24x + 1`

- **س2:** `u = x+1`, `u' = 1`; `v = 2x-3`, `v' = 2`
  - `f' = [1·(2x-3) - (x+1)·2] / (2x-3)² = (2x-3-2x-2)/(2x-3)² = -5/(2x-3)²`

- **س3:** `f = (x² + 4x + 4) - (x² + 4x) = 4` → `f' = 0`
  - (لاحظ: x+2)² و x(x+4) متطابقتان، الفرق ثابت!

- **س4:** `R(x) = 50x - 0.5x²` → `R'(x) = 50 - x`
  - عند `x=30`: `R'(30) = 20` وحدة نقدية
</details>
