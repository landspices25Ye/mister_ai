## 🎯 هدف الدرس

1. اشتقاق معادلات صريحة y (مثلاً `x² + y² = 25`)
2. حساب المشتقة الثانية والثالثة
3. تطبيقات فيزيائية (التسارع)

---

## 📐 الاشتقاق الضمني

### متى نستخدمه؟

عندما لا يمكن حل المعادلة لـ `y` بسهولة (مثلاً `x² + y² = 25` يمكن حلها، لكن `x³ + y³ = 3xy` صعبة).

### الفكرة

نشتق **طرفي** المعادلة بالنسبة لـ `x`، معاملة `y` كدالة في `x`.
- كل `x` نشتقها → `1`
- كل `y` نشتقها → `dy/dx` (نطبق قاعدة السلسلة)
- ثم نحل المعادلة لـ `dy/dx`

---

## ✍️ المثال 1: دائرة

**المطلوب:** جد `dy/dx` إذا كانت `x² + y² = 25`

**الحل:**
1. نشتق طرفي المعادلة بالنسبة لـ x:
```
2x + 2y(dy/dx) = 0
```
2. نحل لـ `dy/dx`:
```
2y(dy/dx) = -2x
dy/dx = -x/y
```

> 💡 **تفسير هندسي:** عند النقطة `(3, 4)` على الدائرة، `dy/dx = -3/4`. هل تتذكر أن ميل المماس عند نقطة على دائرة قطرها رأسي؟

---

## ✍️ المثال 2: معادلة أعقد

**المطلوب:** جد `dy/dx` إذا كانت `x³ + y³ = 3xy`

**الحل:**
1. نشتق:
```
3x² + 3y²(dy/dx) = 3y + 3x(dy/dx)
```
2. نجمع حدود `dy/dx`:
```
3y²(dy/dx) - 3x(dy/dx) = 3y - 3x²
```
3. نخرج `dy/dx` عاملاً مشتركاً:
```
3(dy/dx)(y² - x) = 3(y - x²)
```
4. نقسم:
```
dy/dx = (y - x²) / (y² - x)
```

## 📐 المشتقات العليا

### المفهوم

| الرمز | المعنى |
|---|---|
| `f'(x)` | المشتقة الأولى |
| `f''(x)` أو `d²y/dx²` | المشتقة الثانية |
| `f'''(x)` أو `d³y/dx³` | المشتقة الثالثة |
| `f^(n)(x)` | المشتقة n-ية |

### اشتقاق متتابع

نشتق، ثم نشتق النتيجة، ثم النتيجة، إلخ.

---

## ✍️ مثال: المشتقات العليا

**المطلوب:** جد المشتقات الثلاث الأولى لـ `f(x) = x⁴ - 3x² + 5x - 1`

**الحل:**
- `f'(x) = 4x³ - 6x + 5`
- `f''(x) = 12x² - 6`
- `f'''(x) = 24x`
- `f⁽⁴⁾(x) = 24`
- `f⁽⁵⁾(x) = 0` (وبعدها كلها أصفار)

---

## 🧪 تطبيق فيزيائي

إذا كان `s(t)` الموقع عند الزمن t:
- `v(t) = s'(t)` = **السرعة**
- `a(t) = v'(t) = s''(t)` = **التسارع**

**مثال:** إذا كان `s(t) = 5t³ - 2t` (متر)
- `v(t) = 15t² - 2` (م/ث)
- `a(t) = 30t` (م/ث²)
- عند `t = 2`: `a(2) = 60` م/ث²

---

## 📝 تدريبات

### 🟢 مستوى 1
1. جد `dy/dx` ضمنياً:
   - `x² + y² = 16`
   - `xy = 6`
   - `x² + 4y = 1`

2. جد `f''(x)`:
   - `f(x) = x³ - 6x + 2`
   - `f(x) = sin(x)`
   - `f(x) = e^x`

### 🟡 مستوى 2
3. جد `dy/dx` لـ `x² - y² = 4` (قطع زائد)
4. جد `f'''(x)` لـ `f(x) = x⁵ - 2x³ + x`
5. إذا كان `s(t) = 4t² + t` (متر)، جد السرعة والتسارع عند `t = 3`

### 🔴 مستوى 3
6. جد معادلة المماس لمنحنى `xy = 8` عند النقطة `(2, 4)`
7. إذا كان `s(t) = 16t - 16t²` (متر):
   - متى يصل الجسم لأقصى ارتفاع؟
   - ما أقصى ارتفاع؟ (تلميح: v=0 عند القمة)

## ✅ الإجابات

<summary>اضغط هنا</summary>

**1.**
- `x² + y² = 16` → `2x + 2y(dy/dx) = 0` → `dy/dx = -x/y`
- `xy = 6` → `y + x(dy/dx) = 0` → `dy/dx = -y/x`
- `x² + 4y = 1` → `2x + 4(dy/dx) = 0` → `dy/dx = -x/2`

**2.**
- `f(x) = x³ - 6x + 2` → `f' = 3x² - 6` → `f'' = 6x`
- `f(x) = sin x` → `f' = cos x` → `f'' = -sin x`
- `f(x) = e^x` → `f' = e^x` → `f'' = e^x` (ثابت!)

**3.** `x² - y² = 4`:
   - `2x - 2y(dy/dx) = 0`
   - `dy/dx = x/y`

**4.** `f(x) = x⁵ - 2x³ + x`:
   - `f' = 5x⁴ - 6x² + 1`
   - `f'' = 20x³ - 12x`
   - `f''' = 60x² - 12`

**5.** `s(t) = 4t² + t`:
   - `v(t) = 8t + 1`
   - `a(t) = 8`
   - عند `t=3`: `v(3) = 25` م/ث، `a(3) = 8` م/ث²

**6.** `xy = 8`:
   - `dy/dx = -y/x`
   - عند `(2,4)`: `m = -4/2 = -2`
   - معادلة المماس: `y - 4 = -2(x - 2)` → `y = -2x + 8`

**7.** `s(t) = 16t - 16t²`:
   - `v(t) = 16 - 32t`
   - `v = 0` → `t = 0.5` ثانية
   - `s(0.5) = 16(0.5) - 16(0.25) = 8 - 4 = 4` متر

## 💬 نصيحة المعلم

> الاشتقاق الضمني يبدوا غريباً في البداية، لكن **السر**: عامل `y` كدالة في `x` (لم تنسَ ذلك لأنها ضمنية!). وكل مرة تشتق `y`، ضع `dy/dx` بجانبها. هذا كل ما في الأمر.
