## 📋 الجدول الذهبي (احفظه تماماً!)

| الدالة | المشتقة | اختصار |
|---|---|---|
| `sin(x)` | `cos(x)` | مشتقة الـ sin = cos |
| `cos(x)` | `-sin(x)` | مشتقة الـ cos = **ناقص** sin |
| `tan(x)` | `sec²(x)` | أو `1/cos²(x)` |
| `cot(x)` | `-csc²(x)` | أو `-1/sin²(x)` |
| `sec(x)` | `sec(x)·tan(x)` | مميز! |
| `csc(x)` | `-csc(x)·cot(x)` | مميز! |

### 🧠 طريقة الحفظ (ذكية)

**"sني على c"** (sin → cos) — والجيب والمماس موجبتان دائماً.

**للمشتقات السالبة:** في `cos` و `cot` و `csc`، تأتي بـ **ناقص**.

> 💡 **نصيحة المعلم:** احفظ أول صفين (`sin` و `cos`) فالباقي يُستنتج! لأن:
> - `tan = sin/cos` → بالقسمة
> - `cot = cos/sin` → بالقسمة
> - `sec = 1/cos` → بالقسمة (أو قاعدة السلسلة)
> - `csc = 1/sin` → بالقسمة (أو قاعدة السلسلة)

## ✍️ أمثلة محلولة

### المثال 1: مشتقة sin مع قاعدة القوة

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = 3x² · sin(x)`

**الحل:**
- قاعدة الجداء: `u = 3x²` و `v = sin(x)`
- `u' = 6x` و `v' = cos(x)`
- `f'(x) = 6x · sin(x) + 3x² · cos(x)`

---

### المثال 2: مشتقة tan

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = tan(x) - 3x`

**الحل:**
- `(tan x)' = sec²x`
- `f'(x) = sec²(x) - 3`
- أو بصيغة مكافئة: `f'(x) = 1/cos²(x) - 3`

---

### المثال 3: مشتقة sec

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = x · sec(x)`

**الحل:**
- قاعدة الجداء: `u = x` و `v = sec(x)`
- `u' = 1` و `v' = sec(x)·tan(x)`
- `f'(x) = 1·sec(x) + x·sec(x)·tan(x)`
- `f'(x) = sec(x) · [1 + x·tan(x)]`

---

### المثال 4: قسمة مثلثية

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = sin(x) / (1 + cos(x))`

**الحل:**
- قاعدة القسمة:
- `u = sin(x)`، `u' = cos(x)`
- `v = 1 + cos(x)`، `v' = -sin(x)`
- `f'(x) = [cos(x)·(1+cos(x)) - sin(x)·(-sin(x))] / (1+cos(x))²`
- `= [cos(x) + cos²(x) + sin²(x)] / (1+cos(x))²`
- `= [cos(x) + 1] / (1+cos(x))²` (لأن `cos² + sin² = 1`)
- `= 1 / (1 + cos(x))`

> 🎯 **ملاحظة جميلة:** النتيجة `1/(1+cos(x))`! نتيجتنا مذهلة بعد التبسيط. هذه قوة المثلثات.

## ⚠️ أخطاء شائعة

| ❌ الخطأ | ✅ الصواب |
|---|---|
| `(cos x)' = sin x` | `(cos x)' = -sin x` |
| `(tan x)' = sec x` | `(tan x)' = sec² x` (تربيع!) |
| `(sec x)' = sec x` | `(sec x)' = sec(x)·tan(x)` |
| `(cos 2x)' = sin 2x` | `(cos 2x)' = -2 sin 2x` (تحتاج قاعدة السلسلة) |

## 📝 تدريبات

### 🟢 مستوى 1
1. جد `f'(x)`:
   - `f(x) = 5·sin(x) - 2·cos(x)`
   - `f(x) = x² + 3·tan(x)`
   - `f(x) = 4·cos(x) - x·sin(x)`

2. جد ميل المماس لمنحنى `y = sin(x) + 2cos(x)` عند `x = 0`

### 🟡 مستوى 2
3. جد `f'(x)`:
   - `f(x) = e^x · cos(x)`
   - `f(x) = x · tan(x)`
   - `f(x) = (1 + sin(x)) / cos(x)`

4. أوجد `f'(π/4)` إذا كانت `f(x) = sin(x) · cos(x)`

### 🔴 مستوى 3
5. اشتق: `f(x) = tan(x) · sin(x) + sec(x)` (ستحتاج الجداء والتبسيط)
6. (تطبيقي) حركة توافقية بسيطة: `x(t) = 5·cos(2t)`. جد السرعة والتسارع.

## ✅ الإجابات

<details>
<summary>اضغط هنا</summary>

**1.**
- `f(x) = 5sin x - 2cos x` → `f'(x) = 5cos x + 2sin x`
- `f(x) = x² + 3tan x` → `f'(x) = 2x + 3sec²x`
- `f(x) = 4cos x - x sin x` → `f'(x) = -4sin x - (sin x + x cos x) = -5sin x - x cos x`

**2.** `y' = cos x - 2sin x` → عند `x=0`: `y' = 1 - 0 = 1`

**3.**
- `f(x) = e^x cos x` → `f'(x) = e^x cos x - e^x sin x = e^x (cos x - sin x)`
- `f(x) = x tan x` → `f'(x) = tan x + x sec²x`
- `f(x) = (1+sin x)/cos x`:
  - `u = 1+sin x`, `u' = cos x`; `v = cos x`, `v' = -sin x`
  - `f' = [cos²x - (1+sin x)(-sin x)] / cos²x`
  - `= [cos²x + sin x + sin²x] / cos²x`
  - `= [1 + sin x] / cos²x` (لأن cos² + sin² = 1)
  - `= sec²x + tan x · sec x` (بعد تبسيط إضافي)

**4.** `f(x) = sin x cos x`:
   - `f' = cos²x - sin²x = cos(2x)`
   - عند `π/4`: `cos(π/2) = 0`

**5.** (تحدّي) `f = tan x · sin x + sec x`:
- `f' = sec²x · sin x + tan x · cos x + sec x · tan x`
- `f' = sin x · sec²x + cos x · tan x + sec x · tan x`
- (التبسيط ممكن لكن ليس ضرورياً)

**6.** (فيزيائي):
- `x(t) = 5 cos(2t)` → `v(t) = -10 sin(2t)` → `a(t) = -20 cos(2t)`
</details>

## 🧪 اختبار قصير

**س1:** جد `f'(x)` لـ `f(x) = 2 sin(x) + 5 cos(x) - 3 tan(x)`

**س2:** جد `f'(x)` لـ `f(x) = x² · cos(x)`

**س3:** جد `f'(π/6)` لـ `f(x) = tan(x) · sec(x)`

<details>
<summary>إجابات</summary>

- **س1:** `f'(x) = 2cos(x) - 5sin(x) - 3sec²(x)`
- **س2:** `f'(x) = 2x·cos(x) - x²·sin(x)`
- **س3:** `f'(x) = sec²x · sec x + tan x · sec x tan x = sec³x + sec x · tan²x`
  - عند `π/6`:
    - `sec(π/6) = 2/√3`
    - `tan(π/6) = 1/√3`
    - `sec³ = 8/(3√3)`
    - `sec · tan² = (2/√3)·(1/3) = 2/(3√3)`
    - المجموع: `8/(3√3) + 2/(3√3) = 10/(3√3) = 10√3/9`
</details>
