## 🧠 المخطط الذهني (نظرة عامة)

```
الاشتقاق
├── 1. مفهوم ميل المماس (هندسياً)
├── 2. مفهوم السرعة اللحظية (فيزيائياً)
├── 3. التعريف الرياضي
│   ├── f'(x) = lim(h→0) [f(x+h) - f(x)] / h

│   ├── (c)' = 0  (الثابت)
│   ├── (x)' = 1
│   ├── (x^n)' = n·x^(n-1)  ⭐
│   └── (c·f(x))' = c·f'(x)
└── 5. أمثلة وتطبيقات
```

## 📖 الشرح المفاهيمي

### 🔹 1. ما هي المشتقة؟ (الحدس أولاً)

تخيّل طالباً يقود دراجته على طريق. السرعة اللحظية في لحظة معينة (مثلاً عند الثانية 5) هي:
- المسافة المقطوعة في الثانية الواحدة، **في تلك اللحظة بالذات**

رياضياً: إذا كانت المسافة دالة في الزمن `d(t)`:
```
v(5) = lim(Δt→0) [d(5+Δt) - d(5)] / Δt
```

هذا بالضبط هو مفهوم **المشتقة**: معدل التغيّر اللحظي.

### 🔹 2. التمثيل الهندسي (ميل المماس)

إذا كان عندك منحنى `y = f(x)`، ونريد ميله عند نقطة `(a, f(a))`:
- نرسم **قاطعاً** يمر بنقطتين: `(a, f(a))` و `(a+h, f(a+h))`
- ميل القاطع: `[f(a+h) - f(a)] / h`
- نُقرب `h → 0` (النقطتان تلتقيان)
- نحصل على **ميل المماس** = **f'(a)**

> 💡 **نصيحة المعلم:** المشتقة = "الكاميرا المكبّرة" التي تريك ميل المنحنى في كل نقطة، كأنك تسير على المنحنى بساعة جيب.

### 🔹 3. التعريف الرياضي الرسمي

**الصورة الأولى:**
$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

**الصورة المكافئة (بتغيير المتغير):**
$$f'(x) = \lim_{x_0 \to x} \frac{f(x_0) - f(x)}{x_0 - x}$$

الاثنتان تعطيان نفس النتيجة.

### 🔹 4. الرمز `dy/dx`

مهم جداً أن تعرف أنه:
```
f'(x) = dy/dx = d/dx [f(x)]
```
هذه كلها أسماء لنفس الشيء. نستخدم `dy/dx` كثيراً في التطبيقات.

## 📐 القواعد الأساسية للاشتقاق (احفظها!)

| الدالة | المشتقة | مثال |
|---|---|---|
| `c` (ثابت) | `0` | `5' = 0` |
| `x` | `1` | `x' = 1` |
| `x^n` | `n·x^(n-1)` | `(x^3)' = 3x^2` |
| `c·f(x)` | `c·f'(x)` | `(5x^4)' = 20x^3` |
| `f(x) ± g(x)` | `f'(x) ± g'(x)` | `(x^2 + 3x)' = 2x + 3` |

### ⭐ القاعدة الذهبية: قاعدة القوة

```
إذا كان f(x) = x^n، فإن f'(x) = n · x^(n-1)
```

**أمثلة سريعة:**
- `(x^5)' = 5x^4`
- `(x^10)' = 10x^9`
- `(x^2)' = 2x`
- `(x)' = 1·x^0 = 1`
- `(1)' = 0` (لأن 1 = x^0 و 0·x^(-1) = 0)

**للجذور (تكتبها ككسر أسي):**
- `(√x)' = (x^(1/2))' = (1/2)·x^(-1/2) = 1/(2√x)`
- `(∛x)' = (x^(1/3))' = (1/3)·x^(-2/3)`

## ✍️ أمثلة محلولة (خطوة بخطوة)

### المثال 1: اشتقاق دالة كثيرة حدود بسيطة

**المطلوب:** جد مشتقة `f(x) = 3x^4 - 5x^2 + 7x - 2`

**الحل:**
- نطبّق قاعدة القوة على كل حد:
  - `(3x^4)' = 3·4·x^3 = 12x^3`
  - `(-5x^2)' = -5·2·x = -10x`
  - `(7x)' = 7`
  - `(-2)' = 0`
- نجمع:
```
f'(x) = 12x^3 - 10x + 7
```

✅ **الإجابة:** `f'(x) = 12x^3 - 10x + 7`

---

### المثال 2: اشتقاق دالة تحتوي جذور

**المطلوب:** جد `dy/dx` إذا كانت `y = 2√x + 1/x²`

**الحل:**
- نعيد كتابة الدالة بصيغة الأسي:
  - `2√x = 2·x^(1/2)`
  - `1/x² = x^(-2)`
- الدالة تصبح: `y = 2x^(1/2) + x^(-2)`
- نشتق حدًّا حدًّا:
  - `(2x^(1/2))' = 2·(1/2)·x^(-1/2) = x^(-1/2) = 1/√x`
  - `(x^(-2))' = -2·x^(-3) = -2/x^3`
```
dy/dx = 1/√x - 2/x^3
```

---

### المثال 3: ميل المماس عند نقطة

**المطلوب:** جد ميل المماس لمنحنى `y = x^3 - 2x` عند النقطة `(1, -1)`

**الحل:**
1. جد المشتقة:
```
dy/dx = 3x^2 - 2
```
2. عوّض `x = 1`:
```
dy/dx عند x=1 = 3(1)² - 2 = 3 - 2 = 1
```

✅ **الإجابة:** ميل المماس = `1`

---

### المثال 4: معادلة المماس عند نقطة

**المطلوب:** جد معادلة المماس لمنحنى `y = x² + 1` عند النقطة `(2, 5)`

**الحل:**
1. المشتقة: `dy/dx = 2x`
2. الميل عند `x=2`: `m = 2·2 = 4`
3. معادلة المماس: `y - y₁ = m(x - x₁)`
4. نعوض: `y - 5 = 4(x - 2)`
5. نبسّط: `y = 4x - 3`

✅ **الإجابة:** معادلة المماس: `y = 4x - 3`

## ⚠️ أخطاء شائعة احذر منها

| ❌ الخطأ | ✅ الصواب | لماذا؟ |
|---|---|---|
| `(x^2)' = x^2` | `(x^2)' = 2x` | تنسى طرح 1 من الأسي |
| `(5x^3)' = 5x^3` | `(5x^3)' = 15x^2` | الثابت يبقى في الخارج (يكون ضرب) |
| `(x + 5)' = x + 1` | `(x + 5)' = 1` | مشتقة الثابت 5 هي صفر، لا 1 |
| `(1/x)' = 0` | `(1/x)' = -1/x²` | أعد كتابتها كـ `x^(-1)` أولاً |
| `(√x)' = √x` | `(√x)' = 1/(2√x)` | أعد كتابتها كـ `x^(1/2)` أولاً |

> 💡 **القاعدة الذهبية:** قبل الاشتقاق، أعد كتابة الجذور والكورور المقلوبة بصيغة أسس كسرية. هذا يوفّر عليك أخطاء كثيرة.

## 📝 تدريبات (متدرجة الصعوبة)

### 🟢 مستوى 1 (تأسيسي)
1. جد `f'(x)` للدوال التالية:

2. جد `dy/dx` إذا كانت:
   - `y = x^4 + 3x^2 - x + 10`

### 🟡 مستوى 2 (متوسط)

4. جد معادلة المماس لمنحنى `y = x³ - 3x` عند النقطة `(2, 2)`.
5. جد `dy/dx` إذا كانت:
   - `y = 3√x + 1/x`
   - `y = x^(1/3) - 2x^(-1/2)`

### 🔴 مستوى 3 (تحدّي)
6. أوجد نقطة على منحنى `y = x² - 6x + 5` التي يكون فيها المماس أفقياً (الميل = 0).
7. إذا كانت `s(t) = 5t² + 3t` تعبر عن المسافة بالـ (متر) كدالة في الزمن بالـ (ثانية)، جد:
   - السرعة اللحظية عند t = 4 ثوان
   - متى تكون السرعة = 13 م/ث؟

## ✅ الإجابات

<details>
<summary>اضغط هنا لرؤية الإجابات (بعد المحاولة!)</summary>

**1.**
- `f(x) = 7` → `f'(x) = 0`
- `f(x) = x^5` → `f'(x) = 5x^4`
- `f(x) = 4x^3 - 2x` → `f'(x) = 12x^2 - 2`

**2.**
- `y = x^4 + 3x^2 - x + 10` → `dy/dx = 4x^3 + 6x - 1`
- `y = 5x^6 - x^2 + 7` → `dy/dx = 30x^5 - 2x`

**3.** `y = x² - 4x` → `dy/dx = 2x - 4` → عند `x=1`: `m = -2`

**4.** `y = x³ - 3x` → `dy/dx = 3x² - 3` → عند `x=2`: `m = 9`
   - معادلة المماس: `y - 2 = 9(x - 2)` → `y = 9x - 16`

**5.**
- `y = 3√x + 1/x = 3x^(1/2) + x^(-1)` → `dy/dx = (3/2)·x^(-1/2) - x^(-2) = 3/(2√x) - 1/x²`
- `y = x^(1/3) - 2x^(-1/2)` → `dy/dx = (1/3)x^(-2/3) + x^(-3/2) = 1/(3x^(2/3)) + 1/x^(3/2)`

**6.** نريد `dy/dx = 0`:
   - `2x - 6 = 0` → `x = 3`
   - `y(3) = 9 - 18 + 5 = -4`
   - **النقطة:** `(3, -4)`

**7.** `s(t) = 5t² + 3t` → `v(t) = s'(t) = 10t + 3`
   - عند `t=4`: `v(4) = 43` م/ث
   - نريد `v(t) = 13`: `10t + 3 = 13` → `t = 1` ثانية
