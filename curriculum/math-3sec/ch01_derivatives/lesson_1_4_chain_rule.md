# الدرس 1.4: قاعدة السلسلة (الاشتقاق المركّب)

> **الزمن المقترح:** 60-80 دقيقة
> **الصعوبة:** ⭐⭐⭐⭐
> **متطلبات:** جميع الدروس السابقة

---

## 🎯 هدف الدرس

1. فهم مفهوم "الدالة المركّبة" `f(g(x))`
2. استخدام قاعدة السلسلة بطلاقة
3. اشتقاق دوال مركّبة متنوعة
4. التمييز بين "الداخل" و"الخارج"

---

## 🧠 الحدس: قشّرة البصلة

تخيّل أن لديك دالة `f(u) = sin(u)`، حيث `u = x²` (داخلها دالة أخرى).

لكي تشتق:
1. **شيل الطبقة الخارجية** ← `cos(u)`
2. **اضرب في مشتقة الطبقة الداخلية** ← `2x`
3. **عوّض u** ← `cos(x²)`

هذا تماماً ما تفعله قاعدة السلسلة!

---

## 📐 الصياغة الرسمية

$$[f(g(x))]' = f'(g(x)) \cdot g'(x)$$

**بالرموز الأخرى:**
$$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$$

> 💡 **"اضرب في مشتقة الداخل"** — لا تنسَ أبداً!

---

## 📋 جدول المشتقات (مع السلسلة)

| الدالة | المشتقة |
|---|---|
| `(u^n)'` | `n·u^(n-1) · u'` |
| `(sin u)'` | `cos u · u'` |
| `(cos u)'` | `-sin u · u'` |
| `(tan u)'` | `sec²u · u'` |
| `(e^u)'` | `e^u · u'` |
| `(a^u)'` | `a^u · ln(a) · u'` |
| `(ln u)'` | `u'/u` |
| `(√u)'` | `u'/(2√u)` |

> ⚠️ **لاحظ:** كل مشتقة "خارج" تضربها في `u'` (مشتقة الداخل)

---

## ✍️ أمثلة محلولة

### المثال 1: دالة أسية مركّبة

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = (3x² + 1)^5`

**الحل:**
- **خارج:** `u^5` → `5u^4`
- **داخل:** `u = 3x² + 1` → `u' = 6x`
- نطبّق: `f'(x) = 5(3x²+1)^4 · 6x = 30x(3x²+1)^4`

---

### المثال 2: دالة مثلثية مركّبة

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = sin(2x + 1)`

**الحل:**
- **خارج:** `sin(u)` → `cos(u)`
- **داخل:** `u = 2x + 1` → `u' = 2`
- نطبّق: `f'(x) = cos(2x+1) · 2 = 2cos(2x+1)`

---

### المثال 3: دالة لوغاريتمية

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = ln(x³ + 1)`

**الحل:**
- **خارج:** `ln(u)` → `1/u`
- **داخل:** `u = x³ + 1` → `u' = 3x²`
- نطبّق: `f'(x) = (1/(x³+1)) · 3x² = 3x²/(x³+1)`

---

### المثال 4: دوال مركّبة مزدوجة

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = e^(sin(x))`

**الحل:**
- **خارج:** `e^u` → `e^u`
- **داخل:** `u = sin(x)` → `u' = cos(x)`
- نطبّق: `f'(x) = e^(sin(x)) · cos(x)`

---

### المثال 5: مع الجداء (متقدّم)

**المطلوب:** جد `f'(x)` إذا كانت `f(x) = x² · (x³ + 1)^4`

**الحل:**
- قاعدة الجداء: `u = x²` و `v = (x³+1)^4`
- `u' = 2x`
- `v' = 4(x³+1)³ · 3x² = 12x²(x³+1)³` (سلسلة)
- `f' = 2x · (x³+1)^4 + x² · 12x²(x³+1)³`
- `f' = (x³+1)³ · [2x(x³+1) + 12x⁴]`
- `f' = (x³+1)³ · [2x⁴ + 2x + 12x⁴]`
- `f' = (x³+1)³ · (14x⁴ + 2x)`
- `f' = 2x(x³+1)³(7x³ + 1)`

---

## ⚠️ أخطاء شائعة

| ❌ الخطأ | ✅ الصواب | لماذا؟ |
|---|---|---|
| `(sin 2x)' = cos 2x` | `(sin 2x)' = 2cos 2x` | تحتاج مشتقة الداخل |
| `(e^(2x))' = e^(2x)` | `(e^(2x))' = 2e^(2x)` | مشتقة `2x` هي 2 |
| `(ln 3x)' = 1/(3x) · 0` | `(ln 3x)' = 1/(3x) · 3 = 1/x` | 3x مشتقتها 3 |
| `(cos²x)' = -2cos x` | `(cos²x)' = -2cos x · sin x` | cos²x = (cos x)²، تحتاج سلسلة وقاعدة قوة |

## 📝 تدريبات

### 🟢 مستوى 1
1. جد `f'(x)`:
   - `f(x) = (x² + 1)^3`
   - `f(x) = sin(3x)`
   - `f(x) = e^(5x)`
   - `f(x) = √(x² + 1)`

### 🟡 مستوى 2
2. جد `f'(x)`:
   - `f(x) = cos(x²)`
   - `f(x) = (3x + 1)^(-2)`
   - `f(x) = ln(2x + 5)`
   - `f(x) = 2^(x³)`

3. جد ميل المماس لمنحنى `y = e^(-x)` عند `x = 0`

### 🔴 مستوى 3
4. جد `f'(x)` لـ `f(x) = sin²(3x) + cos²(3x)` (تلميح: بسّط أولاً)
5. جد `f'(x)` لـ `f(x) = ln(sin(x))` (الجواب أنيق!)
6. (تطبيقي) الاضمحلال الإشعاعي: `m(t) = 100 · e^(-0.05t)`. جد `m'(t)`.

---

## ✅ الإجابات

<details>
<summary>اضغط هنا</summary>

**1.**
- `(x²+1)^3` → `3(x²+1)² · 2x = 6x(x²+1)²`
- `sin(3x)` → `3cos(3x)`
- `e^(5x)` → `5e^(5x)`
- `√(x²+1) = (x²+1)^(1/2)` → `(1/2)(x²+1)^(-1/2) · 2x = x/√(x²+1)`

**2.**
- `cos(x²)` → `-sin(x²) · 2x = -2x sin(x²)`
- `(3x+1)^(-2)` → `-2(3x+1)^(-3) · 3 = -6/(3x+1)³`
- `ln(2x+5)` → `2/(2x+5)`
- `2^(x³)` → `2^(x³) · ln(2) · 3x² = 3x² · ln(2) · 2^(x³)`

**3.** `y = e^(-x)` → `y' = -e^(-x)` → عند `x=0`: `y' = -1`

**4.** `sin²(3x) + cos²(3x) = 1` (متطابقة) → `f' = 0`

**5.** `f(x) = ln(sin x)`:
   - خارج: `ln u` → `1/u`
   - داخل: `u = sin x` → `u' = cos x`
   - `f' = (1/sin x) · cos x = cos x / sin x = cot x`
   - **النتيجة الجميلة:** `f'(x) = cot(x)`

**6.** `m(t) = 100 e^(-0.05t)`:
   - `m'(t) = 100 · e^(-0.05t) · (-0.05) = -5 e^(-0.05t)`
   - هذا هو "معدل الاضمحلال اللحظي"
</details>

## 🧪 اختبار قصير

**س1:** جد `f'(x)` لـ `f(x) = (x + 1)^5`

**س2:** جد `f'(x)` لـ `f(x) = e^(2x²)`

**س3:** جد `f'(x)` لـ `f(x) = sin(ln x)` (الجواب أنيق!)

**س4 (تطبيقي):** تكلفة الإنتاج `C(x) = (500 + 3x)²`. جد التكلفة الحدية عند `x = 10`.

<details>
<summary>إجابات</summary>

- **س1:** `f'(x) = 5(x+1)^4`
- **س2:** `f'(x) = e^(2x²) · 4x = 4x · e^(2x²)`
- **س3:** `f'(x) = cos(ln x) · (1/x) = cos(ln x)/x`
- **س4:** `C'(x) = 2(500+3x) · 3 = 6(500+3x)`
  - عند `x=10`: `C'(10) = 6·530 = 3180`
</details>

## 💬 نصيحة المعلم

> قاعدة السلسلة هي "المفتاح" الذي يفتح لك كل الدوال. عندما ترى دالة داخلها دالة، تذكّر: "**خارج × مشتقة الداخل**". ولكي تكتسب طلاقة، تدرّب على تمييز "الداخل" أولاً قبل أن تبدأ بالاشتقاق. هذا التمرين الذهني أسرع من الحفظ.
