## الفصل 5-6: المصفوفات والمحددات

### المفاهيم الأساسية

**المصفوفة:** ترتيب مستطيل من الأرقام
```
A = [a b]
    [c d]
```

**رتبة المصفوفة (Order):** m × n (m صفوف، n أعمدة)

**أنواع خاصة:**
- **مصفوفة مربعة:** m = n
- **مصفوفة قطرية:** العناصر خارج القطر الرئيسي = 0
- **مصفوفة الوحدة (I):** قطرية، أقطارها 1
- **مصفوفة صفرية:** كلها أصفار
- **مصفوفة منقولة (Aᵀ):** تبديل الصفوف بالأعمدة

**العمليات:**
- **الجمع:** A + B (نفس الرتبة، جمع عنصر-بعنصر)
- **الضرب بعدد:** c·A (ضرب كل عنصر في c)
- **ضرب المصفوفات:** A·B (يجب أن يكون عدد أعمدة A = عدد صفوف B)
  - `(AB)ᵢⱼ = Σ Aᵢₖ·Bₖⱼ`
  - **غير تبديلي:** AB ≠ BA عموماً

**المحددات:**
- 2×2: `det = ad - bc`
- 3×3: قاعدة ساروس أو التوسيع
- خصائص: تبادل صفين يغير الإشارة، عامل مشترك من صف، الخ

**المصفوفة المعكوسة:**
- `A·A⁻¹ = A⁻¹·A = I`
- طريقة: المصفوفة الموسعة [A | I] → Gauss-Jordan → [I | A⁻¹]
- الشرط: det(A) ≠ 0

## الفصل 8: المتجهات في الفراغ

**المتجه في 3D:** `v = <a, b, c>`

**العمليات:**
- **الجمع:** `<a,b,c> + <d,e,f> = <a+d, b+e, c+f>`
- **الضرب بعدد:** `k·<a,b,c> = <ka, kb, kc>`
- **المقدار (Norm):** `|v| = √(a²+b²+c²)`
- **المتجه الوحدة:** `u = v/|v|`

**الضرب القياسي (Dot Product):**
`v·w = a·d + b·e + c·f`
- نتيجة = عدد (وليس متجهاً)
- `v·w = |v||w|cos θ`
- **استخدامات:**
  - إيجاد الزاوية: `cos θ = v·w / (|v||w|)`
  - الإسقاط: `proj_w(v) = (v·w/|w|²)w`
  - التعامد: `v ⊥ w ↔ v·w = 0`

**الضرب الاتجاهي (Cross Product):**
- نتيجة = متجه
- `v × w = <bf-ce, cd-af, ae-bd>`
- **خصائص:**
  - `v × w ⊥ v` و `v × w ⊥ w`
  - `|v × w| = |v||w|sin θ` = مساحة متوازي الأضلاع
  - `v × w = -(w × v)`

**معادلة المستوي:**
`n·(r - r₀) = 0` → `ax + by + cz = d`

**معادلة المستقيم في الفراغ:**
- المتجهة: `r = r₀ + tv`
- المتماثلة: `(x-x₀)/a = (y-y₀)/b = (z-z₀)/c`

## الفصل 10-11: الاحتمالات والإحصاء

### المتغيرات العشوائية المنفصلة
- التوزيع الاحتمالي p(x)
- **التوقع (Mean):** `μ = Σ x·p(x)`
- **التباين (Variance):** `σ² = Σ (x-μ)²·p(x)`
- **الانحراف المعياري:** `σ = √σ²`

### التوزيع الطبيعي
- منحنى الجرس
- `f(x) = (1/(σ√(2π))) e^(-(x-μ)²/(2σ²))`
- خواص:
  - متماثل حول μ
  - المساحة الكلية = 1
  - تقريباً 68% في [μ-σ, μ+σ]
  - تقريباً 95% في [μ-2σ, μ+2σ]
  - تقريباً 99.7% في [μ-3σ, μ+3σ] (قاعدة 68-95-99.7)

### التوزيع ذي الحدين
- P(X=k) = C(n,k) p^k (1-p)^(n-k)
- μ = np، σ² = np(1-p)

### التقدير
- **فترة ثقة للمتوسط:** `x̄ ± z·(σ/√n)`
- فترة ثقة للنسبة: `p̂ ± z·√(p̂(1-p̂)/n)`

### اختبار الفرضيات
- H₀ (فرضية العدم) و H₁ (الفرضية البديلة)
- قيمة p
- مستوى الدلالة α
- قرار: نرفض H₀ إذا p < α

## الفصل 12: المتتاليات والمتسلسلات

### المتتاليات الحسابية
- **aₙ = a₁ + (n-1)d**
- **المجموع:** Sₙ = (n/2)(a₁ + aₙ) = (n/2)(2a₁ + (n-1)d)

### المتتاليات الهندسية
- **aₙ = a₁ · r^(n-1)**
- **المجموع:** Sₙ = a₁(1-rⁿ)/(1-r) حيث r ≠ 1
- **مجموع لا نهائي:** S = a₁/(1-r) إذا |r| < 1

### متسلسلات لا نهائية أخرى
- p-series: Σ(1/n^p) يتقارب إذا p > 1
- المتسلسلة الهندسية: تتقارب إذا |r| < 1
- اختبار النسبة، اختبار الجذر (في كتب الجامعة)
