# هيكل الفصل الثاني: تطبيقات التفاضل
> **الحالة:** هيكل تفصيلي (يحتاج إكمال أمثلة وتدريبات لاحقاً)
> **الزمن الإجمالي للفصل:** 5-7 حصص

---

## الدرس 2.1: المعدلات المرتبطة (Related Rates)

**المفاهيم:**
- معدل تغيّر كمية بدلالة أخرى
- إيجاد معدل تغيّر مجهول من معلوم
- الخطوات الأساسية: (1) ارسم (2) سمّ (3) اكتب المعادلة (4) اشتق بالنسبة للزمن (5) عوّض

**مثال نموذجي:**
ينفخ بالون دائري بحيث يزداد نصف القطر بمعدل 2 سم/ث. جد معدل تغيّر الحجم عند r = 5.
- V = (4/3)πr³ → dV/dt = 4πr² · dr/dt
- عند r=5: dV/dt = 4π(25)(2) = 200π سم³/ث

**التدريبات النمطية:**
1. وعاء مخروطي يمتلئ بالماء — معدل ارتفاع منسوب الماء
2. سلّم يميل على حائط — معدل حركة قاعدته
3. سيارة مبتعدة عن نقطة — معدل تغيّر الزاوية

## الدرس 2.2: القيم القصوى (Extreme Values)

**المفاهيم:**
- القيمة العظمى المطلقة والمحلية
- القيمة الصغرى المطلقة والمحلية
- النظرية: عند القيم القصوى المحلية (في نقطة داخلية)، f'(x) = 0 أو غير موجودة
- اختبار المشتقة الأولى والثانية

**خطوات إيجاد القيم القصوى على [a, b]:**
1. أوجد f'(x) = 0
2. أوجد النقاط الحرجة
3. قيّم f عند النقاط الحرجة + طرفي الفترة
4. الأصغر = صغرى، الأكبر = عظمى

**مثال:**
f(x) = x³ - 3x² - 9x + 5 على [-2, 6]
- f'(x) = 3x² - 6x - 9 = 3(x² - 2x - 3) = 3(x-3)(x+1)
- f'(x) = 0 عند x = 3 و x = -1
- f(-2) = -1, f(-1) = 10, f(3) = -22, f(6) = 59
- **عظمى مطلقة = 59 عند x = 6**، **صغرى مطلقة = -22 عند x = 3**

## الدرس 2.3: التقعّر ونقاط الانعطاف

**المفاهيم:**
- التقعّر لأعلى / لأسفل
- اختبار المشتقة الثانية
- نقاط الانعطاف (Inflection Points)

**الاختبار:**
- f''(x) > 0 → تقعّر لأعلى (U)
- f''(x) < 0 → تقعّر لأسفل (∩)
- f'' يتغير إشارته → نقطة انعطاف

**مثال:**
f(x) = x³ - 6x² + 9x + 2
- f' = 3x² - 12x + 9
- f'' = 6x - 12 = 6(x - 2)
- f'' = 0 عند x = 2
- f''(1) = -6 < 0 (تقعّر لأسفل)
- f''(3) = 6 > 0 (تقعّر لأعلى)
- **نقطة انعطاف: x = 2, y = 4**

## الدرس 2.4: رسم المنحنيات (Curve Sketching)

**الخطوات الكاملة:**
1. **المجال** (Domain)
2. **التقاطعات** مع المحاور
3. **التماثل** (زوجية/فردية)
4. **النهايات** عند حدود المجال (سلوك طرفي)
5. **المشتقة الأولى**: مواقع f'=0، فترات التزايد والتناقص، القيم القصوى
6. **المشتقة الثانية**: التقعّر، نقاط الانعطاف
7. **الرسم** النهائي

## الدرس 2.5: مسائل تطبيقية على القيم القصوى

**الأنواع الكلاسيكية:**
1. **علبة بأبعاد قصوى** — تصنع من ورقة مربعة
2. **أقصر مسافة** بين نقطتين مع قيد
3. **أقصى مساحة** بمستوى محيط ثابت
4. **أقل تكلفة** لإنتاج كمية محددة
5. **أقصى ربح** / **أقل تكلفة وحدة**

**مثال:**
علبة مربعة القاعدة مفتوحة من الأعلى، حجمها 32 دسم³. ما أبعادها بأقل خامة؟
- V = x²·y = 32 → y = 32/x²
- S = x² + 4xy (مساحة السطح)
- S = x² + 4x·(32/x²) = x² + 128/x
- dS/dx = 2x - 128/x² = 0
- 2x³ = 128 → x³ = 64 → x = 4
- y = 32/16 = 2
- **الإجابة: 4×4×2**
